#HCDS 512: HUMAN CENTERED DATA SCIENCE
## A1: DATA CURATION

#### The goal of this repository is to construct, analyze, and publish a dataset of monthly English Wikipedia mobile and desktop page traffic from the earliest month where data is available through the most recent month where data is available.

### API Calls 

1. [The *Legacy Pagecounts API*](https://wikitech.wikimedia.org/wiki/Analytics/AQS/Legacy_Pagecounts)
2. [The *Pageviews API*](https://wikitech.wikimedia.org/wiki/Analytics/AQS/Pageviews)
